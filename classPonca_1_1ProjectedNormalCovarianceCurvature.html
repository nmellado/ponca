Detailed Description

template<class DataPoint, class _WFunctor, typename T>
class Ponca::ProjectedNormalCovarianceCurvature< DataPoint, _WFunctor, T >

Extension to compute curvature values based on a covariance analysis of normal vectors of neighbors projected onto the tangent plane.

A 2D covariance matrix is computed from the projections of normals of the neighbors and the two principal curvature values and directions are given by the eigenvalues and associated eigenvectors of the covariance matrix [2].

Note
This procedure requires two passes, the first one for plane fitting and local frame estimation, and the second one for covariance analysis.
Warning
This class is valid only in 3D.
